# Котоквиз по аналитической химии

Статический сайт-квиз в пастельном розовом стиле с адаптивной версткой для телефона.

## Что внутри

- `index.html` — главная страница.
- `styles.css` — адаптивный дизайн.
- `scripts/questions.js` — база из 298 вопросов.
- `scripts/app.js` — логика теста.
- `assets/kittens-hero.png` — изображение для первого экрана.

## Как опубликовать на GitHub Pages

1. Создайте новый публичный репозиторий на GitHub, например `cute-kitten-quiz`.
2. Загрузите в него все файлы из этой папки.
3. Откройте `Settings` → `Pages`.
4. В `Build and deployment` выберите `Deploy from a branch`.
5. В `Branch` выберите `main` и папку `/root`.
6. Нажмите `Save`.

После публикации сайт будет доступен по адресу:

`https://ваш-логин.github.io/cute-kitten-quiz/`
